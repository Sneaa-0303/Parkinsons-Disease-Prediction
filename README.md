# 🧠 Parkinson's Disease Detection using Machine Learning

## 📌 Overview

This project predicts whether a patient has Parkinson's disease using biomedical voice measurements and machine learning techniques. After preprocessing the data and evaluating different models, the **Random Forest Classifier** achieved the best performance with an accuracy of **92%**.

---

## 📂 Dataset

* Parkinson's Disease Dataset
* 195 samples
* 24 biomedical voice features
* Target: `status` (0 = Healthy, 1 = Parkinson's)

---

## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Scikit-learn
* Matplotlib
* Google Colab

---

## 🤖 Model

* Random Forest Classifier

**Final Accuracy:** **92%**

Evaluation Metrics:

* Accuracy Score
* Confusion Matrix
* Classification Report
